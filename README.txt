# Streamlit Web App

This is a Streamlit web application created by Axel Rodrigo for educational purposes.

## Features
- Select stocks and ETFs from NASDAQ and NYSE.
- Download time series, daily returns, variance-covariance, and correlation matrices.
- Optimize portfolios using the Sharpe Ratio.
